This repository contains a customized AI skill and knowledge system for Bryden Koney.



\## Purpose



The Bryden Operating System is designed to help with:



\- Business strategy

\- Finance

\- Marketing

\- Sales

\- Cybersecurity

\- AI automation

\- Product development

\- Real estate planning

\- Retirement planning

\- Personal execution

\## Operating Principle



Skills define how the AI should think.



Knowledge files define what the AI should know.



Dashboard files define what is currently true.



\## Usage



When using Claude, ask the assistant to reference the relevant skill and knowledge file.



Example:



Use the bryden-chief-of-staff skill and knowledge/dashboard.md to create my weekly execution plan.



Example:



Use bryden-cmo and knowledge/bespoke-metal-prints.md to create a 30-day marketing plan for Bespoke Metal Prints.



Example:



Use bryden-cfo, knowledge/finances.md, and retirement-planner to evaluate whether I should prioritize retirement contributions, business reinvestment, or home purchase readiness.



\## Maintenance



Update:



\- dashboard.md weekly

\- current-state.md monthly

\- business files when strategy changes

\- finance file when income, savings, debt, or home-buying goals change

\- cybersecurity-career.md when education, certifications, or job goals change
